# PHP-Project
A basic e-commerce website built using PHP, MYSQL, HTML and CSS

# Overview
This project represents the first phase of a basic responsive and database structure.
The project displays products listing, store information and user page, with a completed layout.
A customer-facing layout with multiple pages

# FEATURES 
- Header and Footer created using PHP for the templates
- Fully designed with home page, about page, shop page, and register/login page
- Database structure completed and texted
- Static 'Placeholder for products list'
- Consistent color and responsive layout
- Home page (index.php) displaying database content

  # DATABASE
  - Stores users registration info
  - Stores product details

  # TECHNOLOGIES USED
    - HTML5
    - CSS
    - PHP
    - MySQL
    - GitHub

 # FILE OVERVIEW     
- index.php -> Homepage with featured product
- about.php -> About page which stores information
- shop.php -> Shop page which displays product
- product.php -> Individual product page with details
- contact.php -> Contact form
- register.php -> login
